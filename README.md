# Brick Breaker

Casse-briques en terminal, zéro dépendance, Node.js pur.

## Prérequis

- Node.js ≥ 14

## Lancement

```bash
node brick-breaker.js
```

Ou via npm :

```bash
npm start
```

## Contrôles

| Touche | Action |
|--------|--------|
| `Espace` | Lancer la balle |
| `←` / `Q` | Déplacer la raquette à gauche |
| `→` / `D` | Déplacer la raquette à droite |
| `R` | Rejouer (après game over) |
| `Ctrl+C` | Quitter |
